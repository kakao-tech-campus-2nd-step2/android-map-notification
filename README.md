# 맵 위치 검색 안드로이드 앱
- 카테캠 Step2 Week6
- 장소 검색 및 장소 저장이 가능한 안드로이드 앱

## 기능
**1. 초기 진입 화면 추가** 
- firebase의 Remote Config 설정
- serviceState 값이 ON_SERVICE 일 때 
  - 지도 화면으로 진입
- serviceState 값이 ON_SERVICE가 아닐 때 
  - serviceMessage를 초기 진입 화면 하단에 표시
  - 지도 화면으로 진입하지 않음
